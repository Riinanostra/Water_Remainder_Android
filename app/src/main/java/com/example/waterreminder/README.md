# waterreminder

Root application package.

## Key entry points
- `MainActivity`: App entry point and Compose setup.

## Package layout
- `data/`: Persistence, networking, repositories.
- `domain/`: Core models and use cases.
- `ui/`: Compose screens and components.
- `worker/`: WorkManager jobs.
- `notification/`: Reminder notifications.
- `util/`: Helpers and configuration.
